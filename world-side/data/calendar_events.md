# Threat-Timing Calendar: Scheduled Events, May 2026 – Nov 2026

**Purpose.** State-aligned and politically-motivated cyber actors deliberately time operations around scheduled real-world events — elections, summits, sentencing dates, anniversaries. This is a forward calendar of high-value windows feeding the Prophet threat-timing forecaster.

**Window.** 2026-05-02 → 2026-11-30.
**Last updated.** 2026-05-02.
**Authoring rule.** Every entry has a date (or range) plus a source URL. Tentative items are tagged `[expected]` or `[TBD]`.

**Motive class legend.**
- `ELEC` — election interference / influence operations
- `DIPL` — diplomatic / summit-timed operations (signaling, espionage, leak ops)
- `ANNIV` — symbolic anniversary triggers
- `LEGAL` — retaliation around indictments / sentencing / extraditions
- `CRIT-INFRA` — stress windows on critical infrastructure
- `FIN` — financial / sanctions / market-timed operations
- `SYMB` — symbolic gatherings, sport, mass-attention windows
